# Contact management full-stack application using Angular. Nest.js & MySQL.

## In here I have created full-stack contact management web application using Angular, Nest.js and MySQL.

## Requirements

* Angular
* Nest.js
* MySQL

## Run this Application

To run this application, 

* clone this repository `git clone https://github.com/dinushchathurya/nestjs-api` 

* Then `nestjs-api`

* Run `npm install` seperately in both backend and frontend directories

* Then run `ng serve` in your front end folder and `npm run start:dev` in your backend folder

* Then you can test your application on Angular front-end
